# NEPALI PROGRAMMING GUIDE (ROMAN SCRIPT)

## PARICHAYA (INTRODUCTION)
Yo guide ma programming ko mukhya concepts haru Nepali bhasa ma Roman script ma byakhya gareko cha. Yo guide le functions, arrays, strings, loops, range, ra promises jasta programming concepts haru lai sajilo tarika le bujhauna ko lagi banayeko cha.

## FUNCTIONS (KARYAHARU)

### Function ko Paribhasa
Function bhaneko code ko euta block ho jasle specific kaam garcha. Function le code lai reuse garna ra organize garna maddat garcha.

```
// Function declaration
from bana addNumbers(a, b) => {
    phirta a + b;
};

// Function call
chapde("Yogfal: " + addNumbers(5, 7));
```

### Arrow Functions
Arrow function chai function lekhne chhoto tarika ho.

```
// Arrow function
from bana multiply = (a, b) => {
    phirta a * b;
};

// Function call
chapde("Gunanfal: " + multiply(4, 6));
```

## ARRAYS (SAMUHA)

### Array ko Paribhasa
Array bhaneko ekai prakar ko data haru ko ordered collection ho.

```
// Array declaration
from bana fruits = ["Syau", "Kera", "Suntala", "Aaru"];

// Array access
chapde("Mero manparne phal: " + fruits[1]);
```

### Array Methods

```
// Array methods
from bana numbers = [1, 2, 3, 4, 5] => {
    // Push - array ko antya ma element thapcha
    numbers.push(6);
    chapde("Push pachhi: " + numbers);
    
    // Pop - array ko antya bata element hataucha
    numbers.pop();
    chapde("Pop pachhi: " + numbers);
    
    // Length - array ko size
    chapde("Array ko length: " + numbers.length);
};
```

## STRINGS (SHABDA HARU)

### String ko Paribhasa
String bhaneko characters haru ko sequence ho.

```
// String declaration
from bana greeting = "Namaste Sansar!" => {
    // String length
    chapde("String ko length: " + greeting.length);
    
    // String methods
    chapde("Uppercase: " + greeting.toUpperCase());
    chapde("Lowercase: " + greeting.toLowerCase());
    
    // String concatenation
    chapde("Jodiyeko: " + greeting + " Kasto cha?");
};
```

### String Manipulation

```
from bana manipulateString = (text) => {
    // Substring
    chapde("Substring: " + text.substring(0, 7));
    
    // Replace
    chapde("Replace: " + text.replace("Sansar", "Nepal"));
    
    // Split
    chapde("Split: " + text.split(" "));
};

manipulateString("Namaste Sansar!");
```

## LOOPS (CHAKRA HARU)

### For Loop
For loop le specific number of times code run garcha.

```
// For loop
from bana forLoopExample = () => {
    for (let i = 0; i < 5; i++) {
        chapde("For loop iteration: " + i);
    }
};

forLoopExample();
```

### While Loop
While loop le condition true bhaye samma code run garcha.

```
// While loop
from bana whileLoopExample = () => {
    let count = 0;
    while (count < 5) {
        chapde("While loop iteration: " + count);
        count++;
    }
};

whileLoopExample();
```

### Do-While Loop
Do-While loop le kam se kam ek choti code run garcha, ani condition check garcha.

```
// Do-While loop
from bana doWhileExample = () => {
    let count = 0;
    do {
        chapde("Do-While loop iteration: " + count);
        count++;
    } while (count < 5);
};

doWhileExample();
```

## RANGE (DAAYRA)

Range le number ko sequence generate garna help garcha.

```
// Range example
from bana rangeExample = () => {
    // Create range from 1 to 10
    let numbers = Array.from({length: 10}, (_, i) => i + 1);
    chapde("Range 1-10: " + numbers);
    
    // Using range in for loop
    for (let i of numbers) {
        if (i % 2 === 0) {
            chapde(i + " yo jor sankya ho");
        } else {
            chapde(i + " yo bijor sankya ho");
        }
    }
};

rangeExample();
```

## PROMISES (PRATIBADDHATA)

Promises le asynchronous operations handle garna maddat garcha.

```
// Promise example
from bana promiseExample = () => {
    chapde("Promise suru bhayo...");
    
    // Create a promise
    let myPromise = new Promise((resolve, reject) => {
        // Simulate async operation
        setTimeout(() => {
            let success = true;
            
            if (success) {
                resolve("Kaam safal bhayo!");
            } else {
                reject("Kaam asafal bhayo!");
            }
        }, 2000); // 2 seconds delay
    });
    
    // Handle promise
    myPromise
        .then((message) => {
            chapde("Promise resolve bhayo: " + message);
        })
        .catch((error) => {
            chapde("Promise reject bhayo: " + error);
        })
        .finally(() => {
            chapde("Promise sakiyo!");
        });
};

promiseExample();
```

## CONDITIONAL STATEMENTS (SARTAHARU)

```
// If-Else example
from bana checkNumber = (num) => {
    if (num > 0) {
        chapde(num + " yo dhanatmak sankya ho");
    } natra {
        if (num < 0) {
            chapde(num + " yo rinatmak sankya ho");
        } natra {
            chapde(num + " yo shunya ho");
        }
    }
};

checkNumber(5);
checkNumber(-3);
checkNumber(0);
```
